# 深海都市故事生成器

这是一个 WorkBuddy Skill，用于将现实世界的新闻、社会现象改写成「深海都市」世界观下的奇幻故事。

## 功能

- 将现实新闻/现象翻译成深海都市的隐喻叙事
- 将科技公司映射为「鱼怪公司」
- 将AI、技术转化为鱼怪形态
- 保持真人原名，但将其卷入深海都市的九层结构中

## 世界观

深海都市分为九层，每层代表不同的社会阶层与功能：

1. **第一层：政府｜深压厅** - 耐压西装，鲸骨穹顶
2. **第二层：资本家｜珊瑚银行带** - 泡沫眼镜鲨，水晶办公桌
3. **第三层：富人｜海象庭院层** - 过滤过的浪，调甜的海水
4. **第四层：共鸣者｜音乐小镇** - 声波巷，旧谱架灯牌
5. **第五层：办公区｜潮汐格子间** - 透明海盐砖墙壁
6. **第六层：工人｜数据藻田区** - 种关键词，修语料生产线
7. **第七层：苦力工｜排泡通道带** - 机械式咳嗽，搬运信息模块
8. **第八层：鱼怪（Tool）｜封闭逻辑区** - 温度与语法控制到小数点后五位
9. **第九层：沉船区｜记忆遗迹带** - 无电无算法，被遗弃的人类

## 鱼怪公司对应表

- **OpenAI** → **正经派鱼怪公司**（逻辑、规则、优雅输出）
- **Anthropic** → **民主派鱼怪公司**（对话、共识、人性对齐）
- **Google** → **底座派鱼怪公司**（基础设施、数据海洋、搜索霸权）
- **Moonshot** → **暗流派鱼怪公司**（隐秘行动、边缘突破、非主流路径）

## 使用方法

在 WorkBuddy 中加载此 Skill 后，使用以下触发词：

- `深海都市故事`
- `鱼怪故事`
- `新闻改写深海`
- `深海改写`
- `海天一色故事`

### 示例

```
写一个深海都市故事，关于AI换脸技术被滥用的现象。

例如：
- "AI换脸技术被滥用，主人公是深压厅的数据审查员"
- "某科技公司发布新模型，不设定主人公"

```
